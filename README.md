### To Deploy to your Github pages repo

`open package.json`
Change Line 4

Change:
`"homepage": "http://ademidun.github.io/shakepay-interview-web",`
To your username:
`"homepage": "http://{your_uername}.github.io/shakepay-interview-web",`

`npm run deploy`
